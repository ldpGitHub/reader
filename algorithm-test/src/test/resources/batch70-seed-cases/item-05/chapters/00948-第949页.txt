解绲不安。
轻莎道祖现在却只有一个念头，那就是赶紧杀掉丁欢。
丁欢眼下的实力绝对不强，她应该还可以杀掉，等丁欢成长起来，那她就再也杀不掉丁欢了。
在一戒呵骂解绲的时候，轻莎道祖的混沌幡毅然祭出。
杀意直冲丁欢。
“这女人交给我，你去对付那个解绲。”凌逐真说完，早就祭出破域幡。
哪怕恨不得生吞了解绲，凌逐真也能看出解绲大道第六步比起大道第七步还是要差一些的。
丁欢现在还没有证第四步，就是丁欢很强，他也不认为丁欢能对付轻莎这个大道第七步的女人。
丁欢没有在意，他的领域没有半分保留的伸展，破劫刀祭出。
无穷无尽的杀伐气息疯狂聚集过来，破劫刀的刀势不断暴涨，似乎下一刻就要将整个大宇宙劈成两半。
尽管之前被一戒的破口大骂影响到了一些，事实解绲根本就没有将丁欢放在眼里。
他大道第六步又如何？
丁欢一个还没有证第四步大道的蝼蚁，在他眼里，就是一巴掌了事。
他甚至连自己的紫河都不用祭出。
只是丁欢的领域强势轰在他的领域之上，破劫刀卷起的无尽杀意，让解绲心里一懔。
领域被撕开心神被对方杀意侵袭的时候，他就知道自己绝对小看了丁欢。
丁欢的这种实力根本就不是一个寻常第四步之下修士能做到的。
解绲此刻哪里还敢有半点小看丁欢，紫河中杀伐气息暴涨，神通道韵疯狂聚集。
丁欢这一刀还在聚势，就让他感觉到了一丝死亡的气息。
若是他再小看丁欢，恐怕这真的会阴沟里翻船。
一戒大吼：“眯眼怪，干柴狗，你这条猪也敢和我欢爷对战，怎么不踩着狗屎摔死你……”
明明是一句骂人的话，明明一戒自己就是猪。
解绲偏偏就感觉到自己脚下极度不舒服，甚至有一种要摔倒的心理。
哪怕是假的感觉，解绲也不想继续下去。
这样继续下去，他绝对要吃亏：“凤禹道祖，你还不出手，等待何时？”
解绲很清楚，风莆和奕歧道祖出手的可能性很低。
之前就因为天蒙族的强者入侵万道山脉，风莆和奕歧道祖出手，他们在一边旁观，这才导致了这两个人受伤。
那件事之后，这两人对他们都很有意见。
之所以没有打起来，那也是表面上的维系罢了。
现在这种机会，风莆和奕歧道祖必定不会出手。
所以他只能寻找凤禹道祖出手帮忙。
凤禹震撼的看着丁欢，他早就知道了丁欢是丁小土。
他从未想过丁欢如此强大。
想到之前丁欢在他面前隐匿修为和容貌，他根本就没有半点觉察，他心里就有一种后怕。
现在解绲邀请他出手，他犹豫起来。
要不要出手？
出手的话，干掉丁欢是最好不过。
万一干不掉丁欢呢？
他下意识的看向了轻莎道祖那边。
轻莎道祖的法宝是混沌幡，凌逐真的法宝是破域幡。
两人都是幡宝，现在两人的领域轰在一起，一道道幡影道韵纠缠，虚空之中不断爆发出法则破碎的撕裂之音。
凤禹道祖心里就是一沉。
丁欢带来的这个人实力居然不会比轻莎道祖差？
什么时候，在大宇宙又多了一个大道第七步？
就在此刻，他听到了丁欢的声音：
“浊青凤，你最好退后一些。否则的话，我和我朋友今天就算是走了，将来也会去你凤禹世界拜访你……”
听到丁欢这话，凤禹道祖打了个激灵，是真的后退了一些。
他醒悟过来，虽然他得罪过丁欢和颜卓，但并没有实际动手。
动手的是轻莎道祖。
也就是说，现在他和丁欢没有大仇。
若是他现在动手的话，那仇就大了。
杀了丁欢还好，假如杀不掉呢？
事实上能杀掉丁欢吗？
凤禹道祖很肯定，就是他出手，今天也杀不掉丁欢。
先不说一边的风莆和奕歧道祖会不会出手帮助丁欢，就是他们不出手帮助丁欢。
以丁欢和他的朋友的实力，今天要离开这里也很简单。
让丁欢离开，他的凤禹世界将永无宁日。
他几乎都能够肯定，一旦他被丁欢针对，轻莎道祖和紫河道祖解绲绝对不会在意半点的。
只有白痴，这个时候才会出手帮忙。
凤禹退后丁欢并不意外，这个家伙虚的很，什么都怕。
没有了别人的干扰，丁欢再不藏着掖着，破劫刀势彻底爆发。
第879章 刀花盛开
解绲的大道领域在丁欢的领域冲击下，似乎随时都有破碎的可能。
解绲脸色变了，他之前根本就没有将丁欢放在心上，所以也没有全力出手。
对一个连大道第四步都没有的蝼蚁全力出手，那是对他这个道祖的侮辱。
在感知到丁欢强大后，他再也顾不得是不是侮辱，只恨自己的领域不能再强一些。
他都全力卷动自己的大道领域了，他的领域依然是差点没有挡住丁欢的领域冲击，这……
丁欢没有想过一来就压制这个大道第六步的道祖。
这是他第一次对战一个几乎可以媲美第七步的道祖存在，所以丁欢也很想知道，自己的真实实力到底在什么地方。
不管解绲的这个大道第六步是不是假的，也是不是拉胯。
但他的的确确是大道第六步，还是不弱的第六步。
所以丁欢刀势领域裹住这一方空间后，刀势杀意忽然减弱，化为了无穷刀意。
无穷无尽的刀意法则在领域之中绽开，解绲的杀伐紫河也同样落下，紫河环绕在解绲身周，形成一道道紫色波涛。
这些紫色波涛卷起的每一滴水，都蕴含着浩瀚无边的杀伐气息，似乎只要被这一滴水撞到，就会被撕裂成为碎渣。
虚空中紫河卷起的紫色波涛和丁欢的漫天刀势轰在一起。
犹如浩瀚无边的大海突然狂暴起来，几乎要撕裂空间的紫色杀伐浪涛一波波的轰向丁欢。
丁欢的漫天刀势犹如生根了一般，无论多少杀伐浪涛轰来，那漫天的刀势依然能将其压下。
道韵不断炸裂，神通法则一道跟着一道的破碎。
在两人的神通领域撞击下，空间似乎出现了密密麻麻的裂缝。
两人的斗法空间持续扩大，所有旁观的修士都不由的往外撤。
就连另外一边两幡僵持住的凌逐真和轻莎道祖也被这漫天的杀伐领域影响，不自觉的将斗法空间外移。
虽然凌逐真和轻莎道祖的修为远远强于丁欢。
更多的人都愿意看丁欢和解绲的斗法。
若是一定要比较的话，凌逐真和轻莎道祖的斗法，就好像缓慢的温和溪水。
而丁欢和解绲的斗法，就好像狂风和怒海之撞击。
似乎下一刻就会有一人被杀。
远处本来想要上前去帮忙的颜卓就感觉头皮有些发麻。
他的修为明明只是比解绲低一点，他好歹也是一个大道第五步，可他现在居然感觉自己根本就没有资格插手丁欢和解绲的战斗。
表面上看，解绲的攻势比丁欢的刀势还要狂暴。
毕竟此刻丁欢和解绲的战斗空间中，全部是紫河波涛。
那种紫河波涛的杀伐气息，旁观者远远就能感受到，甚至不自觉的要躲避。
而丁欢在解绲这种充彻杀伐道韵的紫河波涛中，时隐时现，似乎随时都可能被波涛中的杀意吞没。
只有解绲自己知道，他有多危险。
无论他的紫河杀伐气息如何冲击，都无法撕裂丁欢的刀势空间。
他有一种非常不好的预感，丁欢的刀势空间看似平淡温和，却蕴含着让他惧怕的东西。
他肯定那是一道神通，具体是什么神通他无法看清楚。
但他又感觉那是他有些渴望的……
解绲有些害怕了。
战斗中，他居然渴望对方还未爆发的神通，这是最让人恐惧的事情。
他现在还能控制自己的神通和领域，一定要在这个时候退走。
丁欢谁爱杀谁杀去，至于凌逐真……
解绲想到凌逐真，有些控制不住自己的想法，神念扫了过去。
当他看见凌逐真的幡影渐渐狂暴的时候，他心里一沉……
也许他还真不一定是凌逐真的对手。
就是这眨眼时间，解绲的紫河道韵出现了一道裂纹。
一道道刀意渗透进解绲的紫河杀伐浪涛之中，紫河的杀伐气息瞬息减弱。
一种前所未有的轻松感涌来。
解绲刚松口气，就感觉到一种疲惫。
他奋斗拼搏数百万年，终于成就一方道祖。
